# htmx-in-rust

Sample app built with

- Rust

    - [sqlx](https://github.com/launchbadge/sqlx) - database connection
    - [tera](https://keats.github.io/tera/) - html template
    - [shuttle](https://www.shuttle.rs/) - development platform
- [htmx](https://htmx.org/)
- Porsgresql
